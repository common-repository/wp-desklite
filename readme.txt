A Simple Helpdesk and Support Plugin for WordPress

== Description ==

WP Desklite is a helpdesk and support ticket system. WP Desklite is a simple, easy to use, web-based customer support system. Users can create ticket from the front-end. WP Desklite comes packed with a bunch of features that help you to run a support ticket with no extra cost.


WHAT DOES IT DO?
Check out the features of the plugin:
* Tickets: Users can submit tickets from the front end and the agents responds from the wordpress backend 
* Access Control:  Admin Can Assign support roles who has access to tickets 
* File attachments: Multiple files can be attached to the Tickets by the user and the Admin 
* E-mail notifications: The User and the admin recives email notification when a ticket is created or replied, 
* E-mail Customizations: All the Emails can be customised by the Admin 
* Ticket Types:  Create unlimited Ticket Types. Eg: Bugs, Request, Customization etc
* Multiple Departments: Create Unlimited Departments and Assign department to tickets for proper management
* Admin can edit ticket at any time
* Fully responsive design for front-end and admin 
* Strength & Flexibility
* Easy to Use
* Shortcodes for the My Tickets and New Ticket Form




== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/wp-desklite` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Tickets -> Settings screen to configure the plugin.


== Frequently Asked Questions ==

= What is the Shortcode to Show the Tickets Form in the Front End  =

[wpdl_my_tickets]
The Above Shortcode can be used to Show Users Tickets in the Front End 
Dont Worry, Normally We add a page automatically when the plugin is added called My Tickets with this shortcode

= What is the Shortcode to Show the Add Tickets Form in the Front End  =
[wpdl_add_ticket]
Use The Above Shortcode to show the Add ticket from in the Front End
